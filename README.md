# Lista de Tarefas (To-Do List)

Uma aplicação simples, intuitiva e responsiva de gerenciamento de tarefas diárias, desenvolvida para ajudar na organização de rotinas e produtividade.

---

## Funcionalidades Planejadas / Implementadas

* **Adicionar Tarefas:** Criação rápida de novos itens na lista.
* **Marcar como Concluída:** Feedback visual para tarefas que já foram finalizadas.
* **Remover Tarefas:** Exclusão de itens da lista de forma simples.
* **Responsividade:** Interface adaptável para dispositivos móveis e desktop.
* *(Opcional)* **Persistência de Dados:** Salvamento automático das tarefas no navegador (LocalStorage) para não perder os dados ao atualizar a página.

---

## Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

* **HTML5:** Estruturação semântica da página.
* **CSS3 / Tailwind CSS:** Estilização moderna, layout limpo e responsivo.
* **JavaScript:** Lógica de manipulação do DOM, criação dinâmica dos elementos da lista e gerenciamento dos estados das tarefas.
